Getting Started

SecureDispatch is a powerful and secure Node.js module for sending one-time passwords (OTPs) via email. It provides a simple and reliable way to deliver OTPs to users for authentication and verification purposes.

Installation Guide

To install SecureDispatch, you can use npm:

npm install secure-dispatch

Quick Start Guide

Here's a quick example of how to use SecureDispatch to send an OTP via email:

const sendOTP = require('secure-dispatch');
                
                    const email = 'user@example.com';
                    const senderEmail = 'your_email@gmail.com';
                    const senderPassword = 'your_password';
                
                    sendOTP(email, senderEmail, senderPassword)
                        .then(otp => {
                            console.log("OTP sent successfully:", otp);
                        })
                        .catch(err => {
                            console.error("Error sending OTP:", err);
                        });

Contributing

How to Contribute

We welcome contributions from the community to help improve SecureDispatch. To contribute, follow these steps:

  1. Fork the repository.
  2. Clone your fork to your local machine.
  3. Create a new branch for your contributions.
  4. Make your changes and test them locally.
  5. Commit your changes and push them to your fork.
  6. Create a pull request with a descriptive title and detailed description.

Pull Request Process

All pull requests will undergo code review before being merged. Please ensure that your code follows our coding guidelines and conventions.
